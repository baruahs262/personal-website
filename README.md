# Shaurya Baruah - Personal Portfolio Website

A modern, sleek personal portfolio website featuring a dark theme and contemporary design.

## Features

- **Modern Design**: Sleek dark theme with smooth animations and transitions
- **Responsive Layout**: Fully responsive design that works on all devices
- **Project Showcase**: Dedicated pages for each major project and experience
- **Easy Navigation**: Clean, intuitive navigation structure

## Pages

- **Home** (`index.html`) - Landing page with hero section, work showcase, and featured projects
- **About** (`about.html`) - Personal background, skills, and contact information
- **ROOKI** (`rooki.html`) - Startup project details
- **Integreon** (`integreon.html`) - Internship experience
- **CCOM - UNH Durham** (`ccom.html`) - Research internship details
- **Nexus Podcast** (`podcast.html`) - Podcast information and episodes

## Getting Started

Simply open `index.html` in your web browser to view the website locally.

For a better development experience, you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.

## Customization

### Adding Your Images

Replace the image placeholders with your actual photos:

1. **Headshot**: Add your professional headshot to the hero section in `index.html`
2. **Work Photos**: Replace the three work photo placeholders with images of you in action

### Updating Content

- Edit the HTML files to add your specific experience details
- Update contact links in `about.html` with your actual social media and email
- Add real content for each project page

### Styling

All styles are contained in `styles.css`. The design uses:
- **Fonts**: Inter and Space Grotesk from Google Fonts
- **Color Scheme**: Dark theme with indigo/violet accents
- **Animations**: Smooth transitions and fade-in effects

### JavaScript Interactions

`script.js` includes:
- Smooth scrolling for anchor links
- Navbar scroll effects
- Intersection Observer for fade-in animations
- Parallax effects
- Active navigation link highlighting

## Design Philosophy

The website follows modern design principles:
- **Minimalist**: Clean, uncluttered interface
- **Professional**: Suitable for job applications and networking
- **Engaging**: Smooth animations and interactive elements
- **Accessible**: High contrast and readable typography
