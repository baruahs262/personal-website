About the Internship

At Integreon, I built agentic AI tools using the Microsoft ecosystem to support legal teams in reviewing documents such as NDAs and MSAs. The system automated clause identification and interpretation by referencing client-specific legal playbooks, eliminating manual lookups and repetitive review steps.

These tools increased reviewer productivity by ~40% and significantly streamlined legal workflows.

To maximize adoption, I integrated the agent directly into Microsoft Word via the Agent SDK, allowing legal teams to install and use the tool as a native Word extension, creating a seamless, in-document experience.

Key Contributions

  • Shipped a Microsoft Word add-in + Copilot agent for automated legal clause review
  • Designed prompt flows, fallback logic, and guardrails for GPT-powered clause interpretation
  • Integrated AI workflows into the Microsoft ecosystem (Word extensions, Agent SDK)
  • Built Power BI dashboards and SQL data models to track clause usage and performance metrics
  • Automated ETL pipelines for reporting productivity and system accuracy
  • Wrote internal technical documentation and participated in sprint planning and code reviews
